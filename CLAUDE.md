# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is an **Obsidian vault** — a personal knowledge management system, not a software project. There are no build, test, or lint commands.

The vault is a **Swedish-language D&D campaign wiki** for the "Rime of the Frostmaiden" campaign, organized as a linked knowledge graph using Obsidian wikilinks (`[[Note Name]]`).

## Vault Structure

- **Rime of the Frostmaiden.md** — Root campaign document; main entry point
- **Sessions/** — Per-session recaps
- **Karaktärer/** — Player characters
- **NPC/** — Non-player characters
- **Platser/** — Locations
- **Uppdrag & rykten/** — Quests and rumors
- **Händelser/** — Notable campaign events

## Conventions

- All content is in **Swedish**
- Notes use Obsidian wikilinks: `[[Note Name]]` for internal links
- Deceased characters are marked with `†` in their names
- The `.obsidian/` folder is app configuration — do not edit manually
